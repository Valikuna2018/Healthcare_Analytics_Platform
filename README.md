## Problem Statement

Breast cancer is a serious disease and early detection is very important.  
The goal of this project is to use machine learning to predict if a tumor is malignant (cancer) or benign (not cancer) based on medical measurements.

---

## Project Objectives

- Explore and understand the dataset  
- Visualize important health features  
- Prepare and scale the data  
- Train machine learning models  
- Compare Logistic Regression and Decision Tree  
- Evaluate model performance  

---

## Dataset Description

This project uses the **Breast Cancer Wisconsin dataset** from `scikit-learn`.

The dataset contains:

- 569 samples (patients)
- 30 numerical features (tumor measurements)
- 1 target column:
  - 0 = malignant
  - 1 = benign

Source:  
Scikit-learn built-in dataset (`load_breast_cancer`)
